The toolkit

Open source under BSL 1.1. Free for non-production use.

MeshQL

Java 21 · Jetty 12 · GraphQL + REST

Build data APIs from JSON schemas and GraphQL types. Supports PostgreSQL, MongoDB, SQLite. Federation via internal and external resolvers.

Docs GitHub

Mesher

CLI · Code generation · LLM-assisted

Point at a legacy database, describe the domain, and generate a complete MeshQL anti-corruption layer. Introspect, convert, generate, run.

Docs GitHub

meshobj

TypeScript · npm

Lightweight TypeScript client for MeshQL APIs. Type-safe queries and mutations with zero dependencies.
